Multi-Threading Racer

A C# .NET application that demonstrates qualities of IPC and multi-threading. It showcases how using multiple threads to complete separate tasks may create variability in results. It creates up to three threads to race each other in randomly generating a targeted int value. They will then have to be the thread that manages to correctly set the flag to actually be declared the winner.

Github: Multi-Threading-Racer
